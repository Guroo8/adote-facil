# 1.Quero adicionar os dados do animal a ser entregue para adoção no sistema
## Inicial: O cliente coloca imagem, nome, peso, tamanho, idade, raça, genêro e outras observações necessárias dentro do sistema para serem vísiveis aos outros usuários
## Normal: Uma moderação avalia os dados enviados e aprova ou não aprova a adição do animal no sistema
## O que pode dar errado: Dados não serem válidos(impossíveis ou erro de informação),e ainda sim ser aprovado.
## Outros: Durante a aprovação os dados podem ser modificados pelo cadastrador
## Conclusão: Caso seja aprovado, será vísivel para outros usuários, caso não seja aprovado, será enviada uma notificação para quem cadastrou falando o que deu de errado no cadastro
